
Mesurer la rapidité d'une personne en tapant sur le clavier.

Administrateur : 

	- visualise les scores des utilisateurs
	- insere les paragraphes à taper -> convertis en image
	- bouton ou fonction automatique : afficher les scores de tous les joueurs après les 2h de jeu

	- 3 essais maximum sur 2h 
	- duree de jeu : 5min (compte à rebours)

	- score maximal 100pts
	- une erreur/mot manquant = -0.5pt



	ADMINISTRATEUR :

	JEU :

Base de données :
